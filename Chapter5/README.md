# Chapter 05. 오차역전파법

## 정리

이번 장에서는 오차역전파법을 이용해 신경망의 기울기를 효율적으로 계산하는 방법을 학습했다.

계산 그래프를 통해 순전파와 역전파의 흐름을 이해했다.

순전파는 입력에서 출력 방향으로 계산을 진행하고, 역전파는 출력에서 입력 방향으로 미분값을 전달한다.

곱셈 노드와 덧셈 노드를 구현하고, 사과 가격 예제로 각 변수의 미분값을 계산했다.

Sigmoid, ReLU, Affine, SoftmaxWithLoss 계층을 사용해 신경망을 계층 단위로 구현했다.

각 계층은 forward와 backward 메서드를 가지며, 이를 통해 순전파와 역전파를 수행한다.

2층 신경망을 구현하고 MNIST 데이터를 이용해 오차역전파법 기반 학습을 진행했다.

오차역전파법은 수치 미분보다 훨씬 빠르게 기울기를 계산할 수 있다.
