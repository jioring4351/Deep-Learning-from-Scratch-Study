# Chapter 04. 신경망 학습

## 정리

이번 장에서는 신경망 학습의 기본 구조를 학습했다.

신경망은 손실 함수를 기준으로 성능을 평가하고, 손실값이 작아지는 방향으로 가중치와 편향을 갱신한다.

데이터셋은 훈련 데이터와 시험 데이터로 나누어 사용한다.

평균 제곱 오차와 교차 엔트로피 오차를 구현했다.

분류 문제에서는 주로 교차 엔트로피 오차를 손실 함수로 사용한다.

미니배치 학습은 전체 데이터 중 일부를 무작위로 뽑아 학습하는 방법이다.

수치 미분을 이용해 기울기를 계산하고, 경사하강법으로 매개변수를 갱신했다.

경사하강법에서는 학습률을 이용해 한 번에 이동할 크기를 조절한다.

2층 신경망을 구현하고 MNIST 데이터를 이용해 학습을 진행했다.

실제 학습에서는 계산 속도가 빠른 오차역전파법으로 기울기를 구한다.
