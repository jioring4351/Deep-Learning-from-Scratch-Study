# Chapter 06. 학습 관련 기술들

## 정리

이번 장에서는 신경망 학습을 효율적으로 하기 위한 여러 기법을 학습했다.

매개변수 갱신 방법으로 SGD, Momentum, AdaGrad, Adam을 실습했다.

SGD는 기울기를 이용해 매개변수를 단순하게 갱신하는 방법이다.

Momentum은 이전 이동 방향을 반영하여 더 안정적으로 최솟값을 찾아간다.

AdaGrad는 매개변수별로 학습률을 조정하는 방법이다.

Adam은 Momentum과 AdaGrad의 특징을 결합한 최적화 방법이다.

가중치 초기값으로 Xavier 초기값과 He 초기값을 학습했다.

Xavier 초기값은 sigmoid, tanh에 적합하고, He 초기값은 ReLU에 적합하다.

배치 정규화는 학습을 빠르고 안정적으로 진행하는 데 도움을 준다.

오버피팅을 줄이기 위한 방법으로 가중치 감소와 드롭아웃을 학습했다.

하이퍼파라미터 탐색은 넓은 범위에서 시작해 점점 범위를 좁히는 방식으로 진행한다.
