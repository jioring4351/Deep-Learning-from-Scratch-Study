# Chapter 03. 신경망

## 정리

신경망은 입력층, 은닉층, 출력층으로 구성된다.

각 층에서는 입력값에 가중치를 곱하고 편향을 더한 뒤, 활성화 함수를 적용한다.

활성화 함수는 신경망에 비선형성을 추가하는 역할을 한다.

대표적인 활성화 함수로는 계단 함수, 시그모이드 함수, ReLU 함수가 있다.

NumPy 배열과 행렬 곱을 사용하면 신경망의 순전파를 효율적으로 구현할 수 있다.

출력층에서는 회귀 문제에 항등 함수, 분류 문제에 소프트맥스 함수를 주로 사용한다.

소프트맥스 함수는 각 클래스에 대한 확률을 출력한다.

MNIST 데이터셋을 이용해 이미 학습된 신경망으로 손글씨 숫자를 예측하는 실습을 진행했다.

예측 결과와 실제 정답을 비교하여 모델의 정확도를 계산했다.

배치 처리를 사용하면 여러 데이터를 한 번에 계산할 수 있어 추론 속도를 높일 수 있다.
